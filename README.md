# ZerotoGOAT Feedback

A minimal mobile-first web form that captures audience feedback and POSTs it to the Beacon backend (`/sms` endpoint) via an ngrok tunnel. Single-file vanilla HTML/CSS/JS, no build step, deployed to Vercel with no-store cache headers. Companion to the Beacon multi-agent feedback analyzer — see the main project for the backend pipeline that ingests, clusters, triages, and auto-fixes design-token issues from submissions made here.
